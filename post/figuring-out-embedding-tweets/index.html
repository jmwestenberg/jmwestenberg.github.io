Embedding Tweets in Markdown

Last updated on Apr 29, 2020 1 min read

As an aspiring user of #econtwitter, I figured I would investigate how to embed tweets within a blog post.

Utilizing the hugo academic framework, I can directly put .html code within the body of the blog’s code. Twitter has a nice built in feature to simply extract html code for the tweet. It should look something like this.

<blockquote class="twitter-tweet">
<p lang="en" dir="ltr">
Been experimenting with Hugo Academic and hosting through github. 
Happy to say I finally have an up and running site 
<a href="https://t.co/NK2vLKMo9U">https://t.co/NK2vLKMo9U</a> 
and dare I say it was actually enjoyable to make!</p>&mdash; 
Joe Westenberg (@jmwestenberg) 
<a href="https://twitter.com/jmwestenberg/status/1255621184365920258?ref_src=twsrc%5Etfw">
April 29, 2020</a></blockquote> 
<script async src="https://platform.twitter.com/widgets.js" charset="utf-8">
</script>

When including that code in the blog we get the following:

Been experimenting with Hugo Academic and hosting through github. Happy to say I finally have an up and running site https://t.co/NK2vLKMo9U and dare I say it was actually enjoyable to make!

— Joe Westenberg (@jmwestenberg) April 29, 2020

This was a fairly simple solution and ideally I’d like to figure out how to get something that looks more like a tweet. Supposedly you can also embed a live twitter feed on a personal webpage. Both of these tasks are getting added to another day’s to-do list.
